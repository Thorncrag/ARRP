# Change Audit Log

This page is the cumulative log for project-wide Change Audits. It preserves governing-rule, template, scoring, and structural-consistency changes that affect how issue pages, audit histories, inventories, and GitHub Project fields should be interpreted.

Proposal-specific Change Audits are recorded in the affected proposal's sibling audit-history file, not duplicated here, unless they also change a governing rule, template, schema, rubric, or cross-project convention.

Related changes made in rapid succession may be collapsed into one coherent log entry when doing so improves readability and still preserves the material changes, affected files or proposal, score or rebaseline effect, unresolved findings, and reason the edits are treated as one change set.

Current cross-issue score, audit-status, and next-audit tracking lives in the [ARRP GitHub Project](https://github.com/users/Thorncrag/projects/2). Change Audit rules are maintained in [Project Methodology](METHODOLOGY.md#change-audit).
