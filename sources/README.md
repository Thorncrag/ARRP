# Locally Retained External Sources

This directory contains external material not created by ARRP and retained locally for citation support, verification, preservation, or backup. Typical contents include government reports, court filings, raw downloaded data, and backup copies of externally published sources.

The presence of a citation in [`../inventory/sources.csv`](../inventory/sources.csv) does not require a local copy. Retain external material here only when doing so is useful and legally and practically appropriate. The source inventory remains the citation catalog for both locally retained and link-only sources.

This README is the directory's sole ordinary ARRP-created exception. Analyses, crosswalks, catalogs, transformed datasets, and project-generated visualizations belong in [`../research/`](../research/).
